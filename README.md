# Lyryx — Платформа для музыкальных энтузиастов

**Lyryx** — это современное веб-приложение, созданное как аналог Genius в рамках дипломного проекта. Платформа позволяет пользователям искать песни, читать и добавлять аннотации к текстам, общаться и узнавать смысл любимых композиций, в том числе с помощью встроенного искусственного интеллекта.

Дизайн интерфейса выполнен в ярком и оригинальном стиле **Frutiger Aero**, сочетающем глянцевые элементы, градиенты и стеклянные (glassmorphic) полупрозрачные панели.

## Ключевые возможности

- 🎵 **База исполнителей и песен:** Просмотр подробных профилей артистов и их дискографии.
- 📝 **Интерактивные аннотации:** Система выделения текста для просмотра, создания, редактирования и удаления объяснений к строкам песен.
- 👍 **Система репутации:** Оценка пользовательских аннотаций (лайки/дизлайки) с начислением репутационных баллов авторам.
- 🤖 **Интеграция ИИ (AI-Powered):** 
  - **Умные объяснения:** Возможность задать ИИ любой вопрос по выделенному фрагменту текста и мгновенно получить ответ.
  - **Умный перевод:** Построчный машинный перевод целых песен с возможностью скрытия/отображения перевода.
- 👤 **Пользовательские профили:** Страницы пользователей с их статистикой, репутацией, историей аннотаций и возможностью редактирования личной информации (имя, био, аватар).
- 🛡️ **Модерация:** Встроенные инструменты для пользователей с ролью `moderator`, позволяющие создавать новых артистов, добавлять песни, загружать обложки и редактировать информацию inline.
- 🔍 **Продвинутый поиск:** Интеллектуальный поиск с результатами по категориям (Артисты, Песни, Пользователи, Совпадения в текстах).
- 🌍 **Мультиязычность:** Полностью переведенный на русский язык интерфейс.

## Технологический стек фронтенда

- **Фреймворк:** [Next.js](https://nextjs.org/) (App Router, React 18)
- **Язык:** [TypeScript](https://www.typescriptlang.org/)
- **Стилизация:** [Tailwind CSS](https://tailwindcss.com/) + tailwindcss-animate
- **Работа с данными:** [SWR](https://swr.vercel.app/) (кеширование, бесконечные списки)
- **Управление формами:** [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/) (валидация)
- **Уведомления:** [react-hot-toast](https://react-hot-toast.com/)
- **Тестирование:** [Vitest](https://vitest.dev/) + React Testing Library (TDD подход, >180 тестов)

## Установка и запуск

1. Убедитесь, что у вас установлен **Node.js 20 LTS** или выше.
2. Клонируйте репозиторий и установите зависимости:

```bash
npm install
```

3. Запустите dev-сервер:

```bash
npm run dev
```

4. Откройте в браузере `http://localhost:3000`.

## Скрипты проекта

```bash
npm run dev     # запуск сервера для разработки
npm run build   # сборка проекта для продакшена
npm run start   # запуск собранного production сервера
npm run lint    # проверка кода линтером
npm run test    # запуск всех unit-тестов через Vitest
```
