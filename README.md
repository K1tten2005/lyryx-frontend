# lyryx-frontend

Первичный фронтенд-скелет на **Next.js** для аналога Genius.

## Что нужно установить

1. **Node.js 20 LTS** (рекомендуется)
2. **npm** (идет вместе с Node.js)

Проверка:

```bash
node -v
npm -v
```

## Как запустить

1. Установить зависимости:

```bash
npm install
```

2. Запустить dev-сервер:

```bash
npm run dev
```

3. Открыть в браузере:

```text
http://localhost:3000
```

Вы увидите страницу-заглушку с текстом, что приложение успешно запущено.

## Полезные команды

```bash
npm run build   # production build
npm run start   # запуск production сервера (после build)
npm run lint    # линтер
```
