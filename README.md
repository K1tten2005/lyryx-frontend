# Lyryx Frontend

Данный репозиторий содержит фронтенд-часть дипломного проекта (аналог платформы Genius для просмотра и аннотирования текстов песен).

## Требования

- Node.js 20 LTS или выше
- npm

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер для разработки:

```bash
npm run dev
```

3. Приложение будет доступно в браузере по адресу: `http://localhost:3000`.

## Основные скрипты

- `npm run dev` — запуск сервера для разработки
- `npm run build` — сборка проекта для продакшена
- `npm run start` — запуск собранного production сервера
- `npm run test` — запуск unit-тестов
- `npm run lint` — проверка кода линтером
