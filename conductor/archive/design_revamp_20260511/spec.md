# Specification: Design Revamp (Original & Premium)

## Overview
The goal is to move away from a "generic AI/Bootstrap" look to a more original, high-quality design that feels premium and literary. The design will combine Brutalist/Modernist elements (boldness, clarity) with Premium Minimalism (sophistication, whitespace), using "Elegant Serif" typography to emphasize the "text/poetry" nature of lyrics.

## Functional Requirements
- **Unified Visual Identity:** Implement a consistent design system across all pages.
- **Enhanced Lyric Display:** Redesign the lyric reading and annotation interface to feel like a high-end publication.
- **Custom Component Library:** Replace generic components (buttons, inputs, modals) with custom-styled elements that reflect the new aesthetic.
- **Homepage Transformation:** Redesign the landing experience to be visually striking and "original".

## Non-Functional Requirements
- **Performance:** Ensure new styles and fonts do not significantly degrade LCP or other performance metrics.
- **Responsiveness:** The "premium" feel must translate seamlessly to mobile.
- **Accessibility:** Maintain high contrast and readable font sizes despite the "aesthetic" focus.

## Acceptance Criteria
- [ ] Site no longer feels "generic" or "second-rate".
- [ ] Elegant Serif typography is used for main content (lyrics, headings).
- [ ] Purple accent color is integrated into the new minimalist/brutalist theme.
- [ ] All components (buttons, inputs, cards) have a unique, custom feel.
- [ ] Homepage and Lyric pages are fully redesigned.

## Out of Scope
- Backend logic changes (unless required for display).
- New features not related to visual presentation (e.g., social feeds).
