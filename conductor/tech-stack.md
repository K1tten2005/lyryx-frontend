# Tech Stack

## Frontend Framework
- **Next.js:** React framework for building server-rendered and statically generated applications.
- **React:** Core library for building user interfaces.

## Language
- **TypeScript:** Typed superset of JavaScript for enhanced code reliability and developer experience.

## Styling
- **Tailwind CSS:** Utility-first CSS framework for rapid and consistent UI development.

## Testing
- **Vitest:** Blazing fast unit test framework.
- **React Testing Library:** For testing React components in a user-centric way.
